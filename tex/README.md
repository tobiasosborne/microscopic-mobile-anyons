# TeX Layout

| ID | Assumption | Type | Status |
|----|------------|------|--------|
| A1 | LaTeX sources mirror markdown documents and use bd-tracked tasks for conversions | Technical | assumed |

Structure:
- `tex/sections/`: main paper sections.
- `tex/appendices/`: appendices (e.g., code listings).
- `tex/figures/`: figure sources.
- `tex/main.tex`: main paper entry point (includes sections).
